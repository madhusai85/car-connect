# CarConnect - Car Rental Platform (C# + SQL Server)

CarConnect is a simple Car Rental System built using C#, SQL Server, and ADO.NET.

This project includes:
- Customer & Admin Authentication
- Vehicle CRUD Operations
- Reservation Management
- Custom Exception Handling
- NUnit Unit Testing

## Database Tables
- Customer
- Vehicle
- Reservation
- Admin

## Technologies Used
- C# (.NET)
- SQL Server
- ADO.NET
- NUnit

## Project Structure
- entity/model
- dao
- exception
- util
- main

Developed as part of the Hexaware CarConnect Coding Challenge.
